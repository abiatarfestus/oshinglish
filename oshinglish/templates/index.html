This application delivers an English-Oshindonga dictionary, searchable from either language. Translation

This application delivers an English-Oshindonga translation searchable from either language. The translator takes in a text of any length in either language and ouputs the equivalent text translated into the target language. The app uses machine learning techniques to process the text.
